# Lines consist of a triple colon separating a card name and its handler.
# The existing handlers are:
# NEXT - this card should be subsumed into another card's entry, and not handled on its own
# NEXTIMG - as above, but the card has a separate image which should be fetched if images are called for
# SPLIT:[remainder of full card name] - other parts of the card name (delimited by " // ") are each looked up and concatenated onto the card data
# FLIP:[flipped card name] - flipped name is looked up and added onto the current entry as "flip" attributes
# FLIP=###:[flipped card name] - as above, but also set the flipid attribute to ###
#
